Brief Project Description
A dynamic Java Web application built with Servlets, JSPs, and Apache Derby JDBC to handle student profile records. 
The system utilizes a Model-View-Controller (MVC) architectural design patterns to ensure modularity and ease of data transfer across sessions.

---

List of Implemented Features
* **CRUD Operations:** Capability to capture, view, update, and remove profiles cleanly from an integrated dashboard.
* **Error Handling:**  System blocks and displays an error when a the same value of primary key are added.
